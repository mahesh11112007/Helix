# Kiraak UI Design System

## Mission

Design every page as a premium AI SaaS application with a clean, modern, and highly polished interface.

## Design Principles

- Prioritize simplicity and readability.
- Maintain a consistent design language across all pages.
- Mobile-first responsive layouts.
- Use an 8px spacing system.
- Use rounded corners (12–16px).
- Apply subtle shadows instead of heavy borders.
- Keep animations smooth and purposeful (150–250ms).
- Avoid clutter.

## Inspiration

- ChatGPT
- Linear
- Notion
- Raycast
- Obsidian

## Typography

- **Font**: Inter (loaded from Google Fonts).
- Large page titles.
- Clear hierarchy (`h1` > `h2` > `h3` > body > caption).
- Comfortable line spacing (`1.5–1.7` for body text).

## Components

- Modern sidebar with icons.
- Sticky top navigation.
- Premium cards with subtle elevation.
- Consistent, accessible buttons.
- Beautiful, well-labeled forms.
- Elegant, sortable tables.
- Search bars with keyboard shortcuts.
- Skeleton loading states.
- Meaningful empty states.
- Graceful error states.

## Color System

- Professional **dark mode** (default).
- Professional **light mode**.
- One primary accent color (used sparingly for CTAs and highlights).
- High contrast ratios (WCAG AA minimum).
- Accessible color combinations throughout.

## Animations

- Smooth page transitions.
- Hover effects on interactive elements.
- Loading animations (spinners, skeletons, progress bars).
- Button press feedback.
- Card hover interactions.

## Dashboard Requirements

Every dashboard must include:

- Statistics cards (KPIs).
- Charts where appropriate (line, bar, donut).
- Recent activity feed.
- Quick action buttons.
- AI suggestions panel.
- Responsive grid layout (mobile → desktop).

## Prohibitions

- **Do NOT** generate Bootstrap-looking layouts.
- **Do NOT** overuse glassmorphism.
- **Do NOT** use excessive gradients.
- **Do NOT** use inconsistent spacing.
- **Do NOT** mix multiple design languages.
- **Do NOT** create cluttered interfaces.

## Goal

Every screen must look production-ready and comparable to a modern SaaS product released in 2026.
