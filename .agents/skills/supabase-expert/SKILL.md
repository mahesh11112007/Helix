# Supabase Expert

## Core Principles
1. **Row Level Security (RLS)**: Enforce strict RLS policies on all tables, utilizing `auth.uid()` and helper functions to secure data access at the database level.
2. **Supabase Auth Integration**: Set up and manage authentication flows (email, OAuth, passwordless) using Supabase Auth, capturing auth events via triggers.
3. **Edge Functions**: Deploy performant, serverless Deno-based Edge Functions for custom API endpoints and background tasks.
4. **Storage Buckets**: Structure and secure file storage using Supabase Storage, applying RLS policies directly to buckets and objects.
5. **Realtime & Client SDK**: Implement real-time database subscriptions and write efficient queries using the Supabase Javascript/Python SDK.
