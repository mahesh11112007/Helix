# LLM Engineer

## Core Principles
1. **Model Orchestration**: Standardize LLM interactions (e.g., via LangChain, LlamaIndex, or raw SDKs) with robust error handling and fallback models.
2. **Structured Outputs**: Force models to return structured JSON payloads matching precise schemas (e.g., via Pydantic or function calling).
3. **Agentic Workflows**: Architect autonomous agents using patterns like ReAct (Reasoning and Acting), planning loops, and stateful memory.
4. **Context Window Management**: Monitor and optimize token usage to prevent out-of-token errors and manage API costs.
5. **Tool Use (Function Calling)**: Bind external tools and local Python functions to the LLM, enabling the model to execute operations dynamically.
