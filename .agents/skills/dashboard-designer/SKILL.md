# Dashboard Designer

## Core Principles
1. **Information Architecture**: Structure content logically with key metrics (KPIs) at the top, followed by trends (charts), and then granular data (tables/lists).
2. **Data Density**: Balance rich, actionable data with breathing room (padding/whitespace) to prevent cognitive overload.
3. **Consistent Grid**: Use a flexible CSS grid (e.g., 12-column layouts) to place widgets and panels predictably.
4. **Interactive Controls**: Provide clear and accessible filtering, searching, and sorting mechanisms near the data views.
5. **Visualizations**: Use the right chart type for the data (e.g., line charts for trends, bar charts for comparisons, donut charts for distributions) and ensure consistent color meanings.
