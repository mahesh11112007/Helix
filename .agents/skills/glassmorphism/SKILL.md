# Glassmorphism

## Core Principles
1. **Frosted Glass Backdrop**: Use `backdrop-filter: blur(12px) saturate(180%)` to create a premium frosted-glass depth.
2. **Semi-Transparent Background**: Combine blur with a translucent background, e.g., `rgba(255, 255, 255, 0.45)` for light mode or `rgba(15, 23, 42, 0.65)` for dark mode.
3. **Subtle Borders**: Add thin, semi-transparent borders to define edges, e.g., `border: 1px solid rgba(255, 255, 255, 0.125)`.
4. **Glowing & Soft Shadows**: Use soft, diffuse drop shadows to elevate elements from the background.
5. **Contrast Accessibility**: Ensure text overlaid on glass elements remains highly legible and meets contrast standards.
