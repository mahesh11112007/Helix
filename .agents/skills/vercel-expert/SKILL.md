# Vercel Expert

## Core Principles
1. **Configuration (`vercel.json`)**: Design custom routing, headers, rewrites, and redirects using `vercel.json`.
2. **Serverless & Edge Architectures**: Optimize backend functions to run within Vercel's serverless boundaries, managing execution timeouts and memory allocations.
3. **Environment Variable Management**: Organize environment variables securely across Production, Preview, and Development environments.
4. **Build Optimization**: Minimize output bundle size, leverage static generation where possible, and configure custom build/install commands.
5. **CI/CD & Preview Deployments**: Utilize Vercel's Git integration for automated previews, instant rollbacks, and alias domain routing.
