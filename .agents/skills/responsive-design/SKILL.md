# Responsive Design

## Core Principles
1. **Mobile-First Approach**: Design and style for the smallest screen first, then layer on enhancements for larger screens.
2. **Fluid Grids and Layouts**: Use relative units (e.g., percentages, `fr`, `vw`, `vh`, `rem`) instead of fixed pixel sizes.
3. **Media Queries**: Utilize strategic breakpoints (e.g., 640px, 768px, 1024px, 1280px) to adapt layouts smoothly.
4. **Responsive Media**: Ensure images and videos scale within their containers (`max-width: 100%; height: auto`).
5. **Touch Targets**: Design buttons and links to be easily tappable on mobile devices (minimum 44x44 pixels target size).
