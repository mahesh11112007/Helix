# UX Designer

## Core Principles
1. **User-Centric Workflows**: Design intuitive flows with minimal friction to help users achieve their goals quickly.
2. **Cognitive Load Reduction**: Chunk information, avoid clutter, and guide users using progressive disclosure.
3. **Error Prevention & Recovery**: Provide clear validation, helpful warnings, and easy undo mechanisms.
4. **Feedback Loops**: Offer instant visual or auditory feedback for all user actions (e.g., loaders, toast notifications).
5. **Usability Heuristics**: Follow standard design conventions (e.g., Jakob's Law) so users know how to interact with the interface immediately.
